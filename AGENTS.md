# cloud-price-radar 协作规则

请默认用中文回答，文档默认用中文书写。涉及专业术语时，如必须使用英文，请用括号补充中文说明。

本仓库只服务 VPS 云服务器比价和 GPU 算力租赁比价。页面、导航、文案和数据处理不要重新引入卡网订阅、官网订阅、官方 API、中转 API 等 AI 订阅相关内容。

页面应保持工具属性：顶部导航、标题区、VPS/GPU Tab、筛选区、紧凑表格、分页。不要做营销页，不要添加右侧“当前筛选结果”面板，不要在顶部展示更新记录。

数据更新记录可以保存在本地数据文件或数据库中，但不要作为页面顶部导航或首屏内容展示。官网按钮文案统一使用“官网直达”。

默认部署目标是 Cloudflare Workers + OpenNext。除非用户明确要求，不要执行真实生产部署、不要访问外部账号、不要读取或提交 `.env`、密钥、令牌等敏感信息。

只要修改了本仓库代码，收尾前先运行必要验证，并把本次改动整理成一个本地 commit。
